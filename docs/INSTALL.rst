eappi.theme Installation
------------------------

To install eappi.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``eappi.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        eappi.theme

* Re-run buildout, e.g. with:

    $ ./bin/buildout
